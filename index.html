Project Idea

ARTIFICIAL INTELLIGENCE CHATBOT

Overview

The project will be based on the development of an AI capable of answering users' questions. The aim is to be able to deliver better user experience in the absence of a human. The project will involve programming to create the AI whilst testing it on users to ask for user feedback on what is great about it and possible improvements on the AI. There will be a lot of collaboration with companies to deliver the best AI possible for them.

Motivation

The project will be interesting because it will be a new experience for me personally, dealing with Artificial Intelligence. In this current technological age, we are developing technologies such as Virtual Reality and Artificial Intelligence. A majority of companies nowadays aim to implement AI into their work environment one way or another. Being able to implement an AI in regards to FAQ users may have allows for fast and effective answers to users’ concerns.

Description

Two main features of this service are the AI being able to have a conversation with the user and provide clear and concise information for users.

The AI should be able to have and maintain a conversation with the user as the user may ask multiple questions about it and it is expected to be able to answer all the questions within its domain. If the chatbot was unable to maintain a conversation, it could crash and cause users to become frustrated and annoyed and it can reflect poorly on the company which uses it.

Similarly, the AI must also be able to provide clear and concise information in response to the user’s questions. The AI should be able to pick up a keyword regarding the users’ input and be able to provide information regarding that topic or keyword without any issue. If the information provided is too long and convoluted, the user will end up confused and therefore the AI will be deemed unreliable.

Tools and Technologies

The software we need are coding programs such as Eclipse or IntelliJ which utilise Java as their core programming language. This can be used to code our AI to answer questions or provide information regarding the users’ concerns. When a keyword is entered into the AI chat, the AI will provide information regarding the users’ input to help make things clearer for the individual. The main hardware we need are computers or laptops where we can work on programming the AI. Then servers may be required so the AI can be uploaded onto them.

Skills required

Some skills required are knowledge of coding techniques in Java as it will be the source of the AI program to be implemented. Another skill necessary is the ability to create surveys or interact with the public because while working on the AI, user feedback is important to improve different aspects of the AI. These skills aren’t necessarily hard to find as there are people who work in these environment and have extensive knowledge of what is required to be done.

Outcome

The outcome is to deliver the best service possible to users’ in relation to FAQs implemented via an AI. The AI should be able to answer and provide clear and concise information to users’ questions. The ability for an AI to answer user questions allows for efficient delivery of information within an adequate timeframe for users. With this being implemented, it doesn’t require human personnel to answer users’ questions and its highly convenient to have an AI answer the questions instead.
